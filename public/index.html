Core Capabilities

Everything you need to stay secure

CheckMate gives your team end-to-end visibility into Auth0 vulnerabilities and the tools to act immediately.

Prompt-Driven Scanning

Trigger Auth0 CheckMate security scans with a single natural language prompt. The agent identifies the action, confirms with you, then executes.

Vulnerability Dashboard

Compare scans over time and track risk trends across environments.

Remediation Guidance

Ask the AI Agent about any finding and get actionable remediation recommendations.

Integrated Actions

Push findings into JIRA, Confluence, and Teams without switching context.

Token Vault Security

All connector credentials are protected by Auth0 Token Vault by default.

Monitoring Workflow & Scan Visualisation

Run recurring scans and visualise results through charts and graphs to track open vulnerability trends and compare findings over time.

Multi-Task Chaining

Combine multiple actions into one prompt. The agent parses each task, seeks approval in sequence, and executes them end-to-end automatically.

Human-in-the-Loop Approval

Every agent action requires explicit user approval before execution. The agent always tells you what it's about to do - nothing runs without your sign-off.

Prompt Templates

Pick from a set of pre-built prompt templates in the agent chat to kick off common tasks instantly - no typing required.
